Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main

import (
	"fmt"
)


func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}


func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}


func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```

Ответ:
```
Программа выведет: 
2
1

В test() переменная x именована и возвращается после выполнения defer, который увеличивает x на 1. Изначально x установлена в 1, после defer она становится 2.
В anotherTest() переменная x не именована в возвращаемом значении, поэтому defer увеличивает x, но это изменение не влияет на возвращаемое значение. Изначально x установлена в 1, и именно это значение возвращается.
Defer
defer откладывает выполнение функции до тех пор, пока окружающая функция не вернется. defer-выражения выполняются в обратном порядке их объявления

```
